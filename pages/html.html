Définition et Fonction

HTML, ou HyperText Markup Language, est un langage de balisage utilisé pour créer et structurer des pages web. Il est fondé sur le SGML (Standard Generalized Markup Language) et est maintenu par le W3C (World Wide Web Consortium).HTML permet de définir la structure et le contenu d'une page web en utilisant des balises. Ces balises indiquent au navigateur comment afficher le contenu. Par exemple, la balise <h1> indique un titre principal, tandis que la balise p> indique un paragraphe. Les balises peuvent également inclure des attributs qui fournissent des informations supplémentaires, comme les liens hypertextes avec la balise < a href="url">.

Structure de Base

Les balises HTML sont des éléments encadrés par des chevrons (>). Elles viennent généralement par paires, avec une balise d'ouverture et une balise de fermeture. Par exemple, "h1"Titre "/h1" crée un titre de niveau 1. Certaines balises sont auto-fermantes, comme "img src="image.jpg" alt="Description"" pour insérer une image. Un document HTML typique commence par la déclaration "!DOCTYPE html" qui spécifie la version du HTML utilisée. Ensuite, il y a l'élément racine "html" qui contient deux sections principales : "head" et "body". "head" : Contient des métadonnées sur le document, comme le titre de la page ("title"), les liens vers les feuilles de style CSS ( "link"), et les scripts JavaScript (" script") "body" : Contient le contenu visible de la page web, comme les titres, paragraphes, images, liens, et autres éléments multimédias.

evolution

HTML a connu plusieurs versions depuis sa création. HTML 4.01, publié en 1999, a été une version stable pendant de nombreuses années. XHTML, une reformulation de HTML en XML, a été introduit pour plus de rigueur syntaxique. HTML5, publié en 2014, est la version la plus récente et apporte de nombreuses nouvelles fonctionnalités.HTML est souvent utilisé en conjonction avec d'autres technologies web : CSS (Cascading Style Sheets) : Pour styliser et mettre en forme le contenu HTML. CSS permet de définir les couleurs, les polices, les marges, et bien plus encore. JavaScript : Pour ajouter de l'interactivité et des comportements dynamiques aux pages web. JavaScript permet de manipuler le DOM (Document Object Model), de gérer les événements, et de créer des applications web riches.De nombreux outils et frameworks facilitent le développement avec HTML, comme les éditeurs de code (VSCode, Sublime Text), les frameworks CSS (Bootstrap, Tailwind CSS), et les bibliothèques JavaScript (React, Angular, Vue.js).
